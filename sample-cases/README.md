# Sample Cases

Place your medical case PDFs in this directory for easy access.

## Recommended Sources for Practice Cases

- **NEJM Case Records** (Case Records of the Massachusetts General Hospital)
- **Clinical Problem-Solving** articles (NEJM)
- **BMJ Case Reports**
- **HumanDx** case discussions
- **Morbidity & Mortality** conference cases from your institution

## Tips

- Look for cases that follow a structured format: HPI → Exam → Workup → Diagnosis
- Cases with clear differential diagnoses work best for scoring
- Start with cases in your specialty area, then branch out

## Note

Due to copyright, we cannot include published case PDFs in this repository.
Download cases directly from the journal websites.
